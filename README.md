### 🧠 TextOrigin
3-class text origin classifier: **Human vs AI-generated vs AI-paraphrased**
- Fine-tuned **RoBERTa-base** on 15k samples (fp16)
- 5k+ paraphrased samples synthesized using **T5**
- 6 engineered features: GPT-2 perplexity, burstiness, entropy, AI phrase ratio
- REST API via **FastAPI** with CLI + explainability support
> Python | RoBERTa | T5 | FastAPI | scikit-learn

### 🔐 Cryptography-ML
ML-based PRNG (Pseudo-Random Number Generator) classifier.
- **XGBoost + Random Forest** across 8 generators with 18 statistical features
- Core finding: **NIST SP 800-22 tests are statistically inoperable at short sequence lengths** — ML classifiers remain effective where NIST fails
- IEEE paper in progress
> Python | XGBoost | scikit-learn | Statistical Analysis
